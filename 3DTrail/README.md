# Chatooly Agent

**Build web tools with AI assistance - No coding experience needed!**

---

## Getting Started

### Mac Users
Double-click **`Chatooly Agent (Mac).command`**

### Windows Users
Double-click **`Chatooly Agent (Windows).bat`**

---

That's it! The app will:
1. Install dependencies (first time only)
2. Start the server
3. Open your browser automatically

**Close the Terminal/Command Prompt window to stop the server.**

---

## Requirements

- [Node.js](https://nodejs.org/) (version 18 or higher)
- Google Chrome (recommended) or any modern browser
