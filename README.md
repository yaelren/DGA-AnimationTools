# DGA-AnimationTools

A collection of creative web-based tools for digital graphics and animation.

## Tools

| Tool | Description |
|------|-------------|
| [3DTrail](./3DTrail) | 3D trail animation effects |
| [3DTypeShaper](./3DTypeShaper) | 3D typography shaping tool |
| [2D-DVD](./2D-DVD) | Classic 2D DVD screensaver bouncing logo |
| [3D-DVD](./3D-DVD) | 3D DVD screensaver with Three.js |
| [GridTimeDisplacement](./GridTimeDisplacement) | Video time displacement grid effects |

## Running a Tool

Each tool can be run locally with a simple HTTP server:

```bash
cd <tool-folder>
npm run dev
```
